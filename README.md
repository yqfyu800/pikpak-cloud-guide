# PikPak 이용 방법 완전 정복: 가입부터 토렌트·마그넷 클라우드 다운로드까지, 무료 프리미엄 받는 법과 요금제 비교까지 한 번에 (초대 코드 혜택 포함)

클라우드 서비스 쓰다가 용량 꽉 차면 어떡하죠, 라는 걱정을 해본 적 있다면 이 글이 딱 맞다.

솔직히 말하면 처음 픽팍(PikPak) 얘기 들었을 때는 그냥 또 클라우드 드라이브겠지 싶었다. 구글 드라이브랑 뭐가 다르겠어. 근데 막상 써보고 나서 생각이 좀 바뀌었다. 포인트는 단순히 "저장공간이 크다"가 아니라 **서버가 대신 다운받아준다**는 점이다. 마그넷 링크 던져주면 내 컴퓨터가 아니라 픽팍 서버가 알아서 파일 받아서 클라우드에 쌓아준다. 내 인터넷 속도랑 전혀 상관없이.

이 글에서는 픽팍이 뭔지부터, 가입하는 법, 토렌트·마그넷 다운로드 사용법, 텔레그램 봇 연동, 무료로 프리미엄 받는 방법, 그리고 요금제 비교까지 전부 정리한다.

## 픽팍(PikPak)이 뭔데?

픽팍은 싱가포르 법인 PIKCLOUD PTE. LTD.가 2021년에 출시한 **프라이빗 클라우드 드라이브**다. 중국에서는 서비스가 안 되고, 한국을 포함한 글로벌 시장을 대상으로 운영 중이다.

보통 클라우드 서비스는 내 인터넷 회선으로 업로드·다운로드를 한다. 근데 픽팍은 다르다. **클라우드 다운로드(Cloud Download)** 기능이 핵심인데, 쉽게 말하면 이렇다.

> 내가 마그넷 링크나 토렌트 파일을 픽팍에 던져주면, 픽팍 서버가 그 파일을 받아서 내 개인 클라우드에 올려준다. 나는 그 뒤에 원하는 기기에서 스트리밍하거나 다운받으면 된다.

서버 네트워크가 기업급이라 파일의 80% 이상이 몇 초 안에 받아진다고 한다. 실제로 5GB짜리 영화 한 편이 5초 이내에 완료됐다는 후기가 꽤 많다. 토렌트 시드가 0개인 파일도 찾아서 받아준 사례도 있다.

무료 플랜은 6GB 저장공간이 제공되고, 프리미엄으로 업그레이드하면 **10TB**까지 늘어난다.

지원 플랫폼도 폭넓다. Android, iOS, Windows, macOS, Android TV, 웹 브라우저, 그리고 WebDAV까지 지원한다.

## PikPak 가입 방법: 초대 코드로 무료 혜택 받기

가입 자체는 간단하다.

1. 👉 [PikPak 공식 페이지에서 가입하기](https://bit.ly/PIkpak)를 클릭한다.
2. 이메일 또는 구글 계정으로 가입할 수 있다.
3. 앱을 설치하거나 웹 버전으로 바로 이용 가능하다.

**초대 코드 `74098243`을 입력하거나, 위 링크로 가입하면 무료 프리미엄 체험 혜택을 받을 수 있다.** 위 링크에는 초대 코드가 이미 포함되어 있어서 따로 입력하지 않아도 된다.

초대 코드 입력 위치는 앱 기준으로: 우측 하단 → 리퍼럴 인센티브 프로그램 → '나는 초대받았어요' 클릭 → 코드 `74098243` 입력.

가입 후 첫 다운로드 완료, 확장 프로그램 설치 등 미션을 완료하면 추가 무료 프리미엄 시간도 쌓인다. 처음 사용하는 사람이라면 돈 한 푼 안 쓰고도 프리미엄 기능을 며칠 동안 체험할 수 있다.

## PikPak 이용 방법 ① — 토렌트·마그넷 클라우드 다운로드

픽팍의 핵심 기능이다. 사용법 자체는 생각보다 훨씬 쉽다.

### PC 앱 또는 웹 버전 기준

1. 픽팍 PC 앱 또는 웹 버전([drive.mypikpak.com](https://bit.ly/PIkpak))에 로그인한다.
2. 홈 화면 상단이나 중앙에 있는 **"+ 추가"** 또는 **"클라우드 다운로드"** 버튼을 클릭한다.
3. 마그넷 링크, 토렌트 파일, 혹은 HTTP 링크를 붙여넣기 한다.
4. 저장 폴더를 확인하고 **"초고속 클라우드 다운로드"**를 클릭한다.
5. 픽팍 서버가 알아서 받아준다. 내 컴퓨터나 인터넷 속도와는 무관하게.

완료되면 '지금 재생' 또는 '클라우드에서 보기'를 선택할 수 있다. 로컬 PC로 내려받고 싶으면 파일 우측 점 세 개 클릭 → 'PC에 다운로드'를 선택하면 된다.

### 지원 링크 유형

픽팍은 마그넷·토렌트 외에도 여러 형식을 지원한다.

- **토렌트(Torrent)** 파일
- **마그넷(Magnet) 링크**
- **ED2K 링크**
- **HTTP 직접 링크**
- **SNS 링크**: X(구 트위터), Facebook, TikTok 등의 영상 링크를 붙여넣으면 영상과 이미지를 저장할 수 있다.

### 스마트 필터 설정 (선택)

설정 → 전송 메뉴에서 **"스마트 필터"**를 켜두면, 토렌트 묶음에 포함된 광고성 파일을 자동으로 걸러준다. 영화나 영상 위주로 받는 사람이라면 켜두는 게 편하다.

## PikPak 이용 방법 ② — 브라우저 확장 프로그램으로 원클릭 저장

Chrome이나 Edge를 쓴다면 픽팍 브라우저 확장 프로그램을 설치하면 더 편해진다.

확장 프로그램을 설치하고 나면, 웹 페이지에서 마그넷 링크나 토렌트 다운로드 버튼이 감지될 때 픽팍 아이콘이 뜬다. 클릭 한 번으로 바로 클라우드에 저장된다. 복사-붙여넣기 단계를 생략할 수 있어서 토렌트 사이트를 자주 쓰는 사람에게는 꽤 편리하다.

추가로, 브라우저 확장 프로그램 설치 후 출석 체크 이벤트(한국 한정)에 참여할 수도 있다. 뒤에서 자세히 설명한다.

## PikPak 이용 방법 ③ — 텔레그램 봇 연동

텔레그램을 자주 쓴다면 이 기능이 꽤 유용하다. 텔레그램에서 **@PikPak_Bot**을 찾아서 연동해두면, 마그넷 링크나 파일을 봇에게 보내는 것만으로 픽팍 클라우드에 자동 저장된다.

연동 방법:

1. 텔레그램에서 @PikPak_Bot을 검색해서 열거나 `t.me/PikPak_Bot`으로 접속한다.
2. 픽팍 앱 → 설정 → 텔레그램 Bot 연동을 클릭한다.
3. QR코드 또는 링크로 인증하면 연동 완료.

이후 텔레그램에서 마그넷 링크나 파일을 봇에게 보내면 픽팍 클라우드에 바로 저장된다.

주의: 봇에 다른 픽팍 계정이 연동되어 있으면 저장이 안 되거나 용량 오류가 뜰 수 있다. 이럴 땐 봇에 `/settings`를 보내서 기존 권한을 취소하고 다시 연동하면 된다.

> iOS 앱의 경우 애플 정책상 앱 내에서 직접 링크 추가가 제한되어 있다. iOS 사용자는 웹 버전이나 텔레그램 봇을 통해 링크를 추가하고 앱에서 확인하면 된다.

## PikPak 이용 방법 ④ — WebDAV 연동 (고급)

프리미엄 회원은 WebDAV 기능을 사용할 수 있다. 이 기능을 켜두면 에어익스플로러, 사이버덕, RaiDrive 같은 WebDAV 클라이언트로 픽팍 드라이브에 접속해서 파일을 탐색하고 스트리밍할 수 있다. rclone을 쓰면 Windows 탐색기에 가상 드라이브로 마운트하는 것도 가능하다.

WebDAV 활성화 방법:

1. 픽팍 웹 버전 → 설정 → 실험실 기능 → WebDAV 옵션 활성화
2. 표시되는 연결 주소, 계정명, 비밀번호를 WebDAV 클라이언트에 입력

현재는 **파일 읽기(탐색·재생·다운로드)만 지원**하고, 파일 쓰기·복사·이동은 지원하지 않는다. 그래도 클라우드 파일을 미디어 플레이어에서 직접 스트리밍하고 싶은 사람에겐 충분히 쓸만하다.

## 무료로 프리미엄 받는 방법 (한국 한정 이벤트 포함)

픽팍 프리미엄을 돈 안 쓰고 얻는 루트가 몇 가지 있다.

### 1. 초대 코드로 가입 혜택 받기

👉 [초대 코드 74098243으로 가입하기](https://bit.ly/PIkpak)를 통해 가입하면 무료 프리미엄 체험 기회를 받을 수 있다. 새로 가입한 사람은 5일 무료 프리미엄이 제공된다.

### 2. 출석 체크 이벤트 (한국 한정)

이게 한국 사용자들한테 꽤 인기 있는 방법이다. **3일 연속 출석 체크**를 달성하면 12시간에서 30일 사이 랜덤한 시간만큼 무료 프리미엄을 받을 수 있다. 이 이벤트는 **한국 지역 한정**으로 운영 중이다.

참여 방법:

1. Chrome 또는 Edge에 픽팍 브라우저 확장 프로그램을 설치하고 로그인한다.
2. 확장 프로그램을 열어 우측 상단의 '출석 체크하여 무료 프리미엄 받기' 버튼을 클릭한다.
3. 출석 페이지에서 'Check in' 버튼을 클릭한다.
4. 3일 연속 출석하면 랜덤 보상 획득.

모바일 앱에서는 현재 출석 이벤트를 지원하지 않으니, 반드시 브라우저 확장 프로그램을 통해 참여해야 한다.

### 3. 미션 완료 보상

처음 로그인했을 때 우측 상단 이벤트란에 '미션 완료하여 프리미엄 획득' 항목이 보인다. 회원가입, 첫 다운로드, 확장 프로그램 설치 등 미션을 완료하면 추가 프리미엄 시간을 받을 수 있다. 단, 이 이벤트 창은 열고 닫는 순간 사라지니 미션을 완료하기 전까지는 열어두는 게 좋다.

## PikPak 프리미엄 요금제 비교

한국은 **글로벌 프리미엄** 대상 국가다(선진국 33개국 포함). 글로벌 vs 지역 프리미엄의 차이와 요금을 정리했다.

### 글로벌 프리미엄 vs 지역 프리미엄 차이

| 구분 | 글로벌 프리미엄 | 지역 프리미엄 |
|------|--------------|--------------|
| 대상 국가 | 미국, 한국, 일본, 영국, 독일 등 33개 선진국 | 기타 국가·지역 |
| 다운로드 속도 | 최대 20MB/s (지역 제한 없음) | 최대 8MB/s |
| 저장 용량 | 10TB | 10TB |
| 4K 스트리밍 | ✅ | ✅ |
| 해외 이동 시 | 속도 유지 | 선진국 방문 시 속도 제한 |

픽팍이 IP 주소, 기기 언어, SIM 카드 국가 코드를 종합해서 자동으로 적합한 등급을 보여주기 때문에, 결제 화면에서 굳이 직접 선택하지 않아도 된다.

### 전체 요금제 일람표

| 플랜 | 기간 | 가격 | 구매 링크 |
|------|------|------|-----------|
| 무료 | - | 6GB / 무료 |  [무료로 시작하기](https://bit.ly/PIkpak) |
| 글로벌 프리미엄 (월간) | 1개월 | ₩9,900/월 |  [월간 구독하기](https://mypikpak.com/drive/payment?invitation-code=74098243) |
| 글로벌 프리미엄 (연간) | 12개월 | ₩99,000/년 (월 환산 ₩8,250) |  [연간 구독하기](https://mypikpak.com/drive/payment?invitation-code=74098243) |
| 추가 용량 (10TB) | 1개월 | 별도 구매 가능 |  [용량 추가 구매](https://mypikpak.com/drive/payment/storage?invitation-code=74098243) |

> **참고**: iOS 앱 내 구매는 앱스토어 수수료가 붙어서 더 비쌀 수 있다. 동일한 계정으로 웹사이트(링크 위 참조)에서 구매하면 더 저렴하다. iOS에서는 인앱 결제를 지원하지 않으므로 웹 결제 후 앱에서 확인하면 된다.

## 무료 vs 프리미엄: 뭐가 다른가

솔직히 무료 플랜은 "기능 시험용"이다. 6GB면 영화 한두 편 받으면 금방 찬다.

| 기능 | 무료 | 프리미엄 |
|------|------|---------|
| 저장 용량 | 6GB | 10TB |
| 클라우드 다운로드 속도 | 제한적 | 최대 20MB/s (글로벌) |
| 동영상 온라인 재생 | 제한 (7일 후 불가) | 4K 원본 화질 지원 |
| 로컬 다운로드 속도 | 100KB/s | 제한 없음 |
| WebDAV | ❌ | ✅ |
| 클라우드 압축 해제 | ❌ | ✅ (ZIP, RAR, 7z) |
| 동시 다운로드 작업 | 제한적 | 다중 동시 작업 가능 |

프리미엄이 만료되면 다시 무료 플랜으로 내려가고, 저장 용량 초과 상태에서는 새 파일 추가가 제한된다. 만료 후 7일이 지나면 온라인 재생이 안 되고 다운로드 속도도 100KB/s로 제한된다.

## 픽팍이 잘 맞는 사람, 안 맞는 사람

픽팍 이용 방법을 읽었으면 이게 내한테 맞는 서비스인지 판단하는 게 중요하다.

**이런 분한테 잘 맞는다:**

- 토렌트·마그넷 링크를 자주 쓰는데, 컴퓨터 켜놓고 기다리는 게 귀찮은 사람
- 대용량 파일을 여러 기기에서 접근해야 하는 사람
- 영상 컬렉션을 클라우드에서 스트리밍하고 싶은 사람
- 텔레그램에서 파일 주고받는 일이 많은 사람
- NAS를 쓰고 있지만 전력 소비·소음이 부담스러운 사람

**이런 분한테는 덜 맞는다:**

- 문서 공동 작업, Google Docs 스타일의 협업이 필요한 사람
- 팀 폴더 공유가 주목적인 사람
- 계정을 여러 사람이 공유하려는 사람 (약관 위반, 24시간 서비스 정지 처리됨)

아니다. 픽팍의 무료 체험 코드를 사용해도 체험 기간 종료 후 자동으로 요금이 청구되지 않는다. 픽팍은 사용자 동의 없이 결제하지 않는다.

**Q. 프리미엄 환불이 되나요?**
일반적으로 환불이 안 된다. 단, 잘못 청구된 경우 30일 이내, 구매 후 한 번도 사용하지 않은 경우 24시간 이내에 `support@mypikpak.com`으로 연락하면 처리 가능하다.

**Q. 파일 업로드 제한이 있나요?**
웹 버전은 파일당 4GB, PC 앱은 파일당 20GB 제한이 있다. 하루 최대 1,000개 파일 업로드 가능. 단, 프리미엄 사용자는 클라우드에서 직접 압축 해제 기능을 쓰면 제한을 우회할 수 있다.

**Q. 중국에서 쓸 수 있나요?**
안 된다. 중국 IP에서는 서비스 이용이 불가하다. 중국용으로는 별도 서비스인 FilePax가 운영 중이다.

**Q. 비프리미엄 상태로 오래 미접속하면 어떻게 되나요?**
2026년 3월 1일부터 개정된 약관에 따라, 비회원이 12개월 연속 미로그인 시 계정 삭제 절차가 시작될 수 있다. 이메일 알림을 받고 7일 이내에 로그인하면 절차가 중단된다.
